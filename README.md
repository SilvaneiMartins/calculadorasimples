## :rocket: Tecnologias

Este projeto foi desenvolvido em meus estudos, com o propósito de aprender e aumentar mais meus conhecimentos a montar um layout html, css e javascrypt. Eu utizei as tecnologias logo abaixo:

-  [HTML](https://developer.mozilla.org/pt-BR/docs/Web/HTML)
-  [CSS](https://developer.mozilla.org/pt-BR/docs/Web/CSS/)
-  [Javascrypt](https://www.javascript.com/)

## :information_source: Como usar

Para clonar e executar esta aplicação, você precisa do [Git](https://git-scm.com) e [Yarn](https://yarnpkg.com/) ou maior instalados. Comandos:

```bash
# Clone this repository
$ git clone https://github.com/SilvaneiMartins/calculadorasimples

# Go into the repository
$ cd calculadorasimples

# Install dependencies
$ executar index.html
```
